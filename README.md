## 📌 Project Description

NFT Resume is a decentralized application (dApp) built on the Stellar Soroban smart contract platform that enables users to mint their professional resumes as unique NFTs. Each resume is securely stored on-chain and tied to a wallet address, ensuring authenticity, ownership, and tamper-proof verification.

This project reimagines traditional resumes as **verifiable digital assets**, making them portable, transparent, and trustless.

---

## 🚀 What it does

NFT Resume allows users to:

* Mint their resume as a unique NFT on the Stellar blockchain
* Store structured professional data (name, skills, experience, portfolio)
* Prove ownership of their resume using their wallet
* Share a verifiable, tamper-proof version of their credentials
* Transfer ownership of the resume NFT if needed

---

## ✨ Features

* 🪙 **NFT-Based Identity**
  Each resume is minted as a unique NFT with a distinct token ID

* 🔐 **Secure Ownership**
  Only the wallet owner can mint or transfer their NFT

* 📄 **On-Chain Resume Storage**
  Resume data is stored securely using Soroban smart contracts

* 🔍 **Public Verification**
  Anyone can verify the authenticity and ownership of a resume

* 🔄 **Transferable NFT**
  Resume NFTs can be transferred between users

* ⚡ **Lightweight & Fast**
  Built on Stellar’s efficient and low-cost infrastructure

---

## 🛠️ Tech Stack

* **Blockchain:** Stellar (Soroban Smart Contracts)
* **Language:** Rust (soroban-sdk)
* **Client:** JavaScript (Stellar SDK + Freighter Wallet)
* **Network:** Stellar Testnet

---

## 📦 Smart Contract Functions

* `mint(user, resume)`
  Creates a new NFT Resume and assigns ownership

* `get_resume(id)`
  Retrieves resume data for a given token ID

* `owner_of(id)`
  Returns the wallet address of the NFT owner

* `transfer(from, to, id)`
  Transfers ownership of the NFT Resume
